Student Performance Prediction

An interactive Deep Learning application employing Artificial Neural Networks (ANN) to predict academic student outcomes based on study habits and metrics.

Python TensorFlow Keras Streamlit Scikit-Learn
View Source Code Back to Portfolio

// 1. Project Overview

Student Performance Prediction using ANN is a machine learning application designed to predict whether a student is likely to pass or fail based on core academic and behavioral factors. By evaluating critical indicators such as study hours, attendance percentage, previous marks, assignments completed, and sleep hours, the system offers data-backed academic evaluations. The system utilizes a trained Artificial Neural Network (ANN) model to analyze input data and generate predictions along with a confidence score.

// 2. The Solution

To solve this, I designed and trained an Artificial Neural Network model using TensorFlow and Keras to classify student outcomes. The predictions are evaluated based on basic parameters: study hours, attendance, previous marks, completed assignments, and sleep hours. Furthermore, I created an interactive web interface using Streamlit, enabling educators or students to input details using intuitive sliders and instantly visualize predicted pass/fail outcomes and probability confidence levels.

03. Key Features

Implemented neural network prediction pipelines combined with interactive visualizer systems:

🧠 Neural Network Prediction

Utilizes a dense feedforward Artificial Neural Network (ANN) optimized to accurately evaluate complex academic parameters.

📊 Performance Analysis

Performs multi-dimensional analysis on academic habits to pinpoint areas of concern.

🎯 Pass/Fail Prediction

Provides clean binary classification outputs indicating whether a student is projected to pass or fail courses.

📈 Confidence Score

Visualizes probability confidence levels alongside classifications, signifying model certainty.

📝 Performance Explainer

Interprets predictions to highlight the key behavioral drivers behind predictions.

🎚️ Interactive Sliders

Allows easy fine-tuning of study hours, previous grades, and sleep duration values via UI sliders.

⚡ Real-Time Prediction

Instantly recalculates and renders prediction outputs as user inputs dynamically change.

🌐 Streamlit Interface

Deploys model files on Streamlit UI framework to provide a responsive, highly accessible browser web application.
