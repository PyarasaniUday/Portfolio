// 1. Project Overview

This project implements a Convolutional Neural Network (CNN) based Image Classifier designed to recognize and identify objects in images. The classifier parses color pixels and organizes predictions across 10 distinct CIFAR-10 categories, including automobiles, dogs, cats, airplanes, and birds, providing automated tags for digital images.

// 2. The Solution

I structured a custom deep learning classifier trained using the benchmark CIFAR-10 dataset. The CNN model extracts spatial patterns through successive convolutional, pooling, and dense layers built using TensorFlow and Keras. The resulting model preprocessing pipeline can ingest user images, return class labels, render confidence graphs, and analyze classification statistics.

03. Key Features

Designed with advanced computer vision pipeline features:

🧠 CNN Architecture

Utilizes sequential 2D convolutional and pooling layers optimized to extract high-level visual features.

🖼️ Multi-Class Classification

Categorizes inputs among 10 distinct object classes with softmax probability distributions.

📊 CIFAR-10 Training

Trained and cross-validated using the benchmark 60,000 tiny image CIFAR-10 dataset.

🎯 Object Recognition

Performs spatial feature mapping to distinguish shapes, edges, and object textures.

📈 Accuracy Evaluation

Generates metrics including training curves, confusion matrices, and validation reports.

🔍 Custom Image Input

Enables users to upload custom jpeg/png files to obtain instant computer vision tags.

⚡ Confidence Scores

Displays a confidence percentage bar highlighting the likelihood profile for each predictions.

🤖 Deep Learning Pipeline

Structured preprocessing including normalization, tensor reshaping, and batch normalization layers.
