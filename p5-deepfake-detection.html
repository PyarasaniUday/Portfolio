// 1. Project Overview

The Deepfake Detection System is an AI-powered cybersecurity solution developed by a team of five members at Keshav Memorial College of Engineering as a part of Project School. The system focuses on identifying, tracking, and isolating manipulated facial assets in digital media by combining spatial features from deep convolutional neural networks with the global contextual representation of Vision Transformer (ViT) architectures.

Our objective is to design a highly performant and secure network capability to verify facial media validity (distinguishing between authentic and AI-synthesized objects). By targeting pixel-level manipulation artifacts and anomalous temporal boundaries, this system works as a key defense layer combating digital spoofing, political misinformation, digital fraud, and identity crimes.

02. Problem Statement

Deepfake creation toolsets have advanced significantly, enabling attackers to synthesize hyper-realistic face-swapping media using generative adversarial pipelines. These manipulations are virtually indistinguishable to the human eye, raising severe vulnerabilities in security protocols:

Fake News & Misinformation

Manipulated visual assets targeting figures of influence to spread falsehoods and trigger social crises.

Identity Theft

Forging identities to bypass biometric access networks and bypass verification gates.

Cybercrime & Fraud

Leveraging fake video loops during live virtual meetings to authorize financial and data transfers.

03. Pipeline Workflow

The system executes an automated end-to-end deep learning framework, converting raw video feeds into precise authenticity confidence percentages:

1

Dataset Collection

Importing the comprehensive Celeb-DF dataset containing both authentic celebrity videos and high-fidelity deepfake synthesis targets.

2

Frame Extraction

Utilizing OpenCV pipeline processes to ingest input video files and perform high-speed extraction of target sequence frames.

3

Face Detection (MTCNN)

Processing extracted frames using Multi-task Cascaded Convolutional Networks to map facial boundary bounding boxes and key coordinate locations.

4

Data Preprocessing

Cropping facial bounding boxes, normalizing RGB coordinate distributions, and resizing tensors to target input dimensions.

5

Feature Extraction

Routing normalized face patches through an EfficientNet feature extractor block to capture localized structural face maps.

6

Vision Transformer Analysis

Processing mapped feature coordinates through self-attention layers in a Vision Transformer module to identify global spatial inconsistencies and pixel manipulation boundaries.

7

Classification

Running output transformer tokens through a classification density block to compute final prediction metrics: Real or Fake.

05. Key Features

🛡️ Deepfake Detection

Identifies micro-pattern changes and fake textures to accurately classify real media vs. synthesized deepfakes.

🤖 Vision Transformer-Based

Leverages self-attention coordinates and multi-head sequence processing to find inconsistent blending borders.

🖼️ Facial Image Processing

Standardizes facial dimensions and normalizes contrast ranges to prepare image inputs for feature models.

🎥 Video Frame Analysis

Scans continuous video frames to analyze temporal anomalies and detect structural variations between sequential clips.

🔍 Face Detection via MTCNN

Isolates human faces in real-time, even in multiple-face scenarios or under complex angles.

⚡ Real-Time Pipeline

Built with optimized GPU data layers to run model evaluation rapidly over target frame sequences.

📊 Confidence Prediction

Yields detailed confidence scores detailing fake-synthesized probabilities for security auditing audits.

🧠 Deep Learning Powered

Combines convolutional architectures with attention networks to ensure strong detection results.

// Celeb-DF Dataset

The model is trained and validated using the Celeb-DF Dataset, which is one of the standard high-quality benchmarks for deepfake detection evaluations.

It offers face clips containing subtle artificial boundaries, making it a challenging and realistic collection for evaluating modern neural classifiers.

Dataset Categories

  • Celeb-Real Authentic
  • YouTube-Real Authentic
  • Celeb-Synthesis AI Manipulated
